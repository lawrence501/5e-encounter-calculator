# 5e-encounter-calculator
Encounter calculator for 5e (using A5E numbers)

## Rules

### Variables

TCL (total character level) = sum of levels of all party members
ECR (encounter challenge rating) = sum of CRs of all monsters

### Difficulty Rules

- Easy: ECR = TCL/6
- Medium: ECR = TCL/3
- Hard: ECR = TCL/2
- Deadly: ECR = (2*TCL)/3
